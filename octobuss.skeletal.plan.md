# Octobuss Skeletal (from scratch)

## Что сделано
- Новый рендер с нуля в `octobuss.skeletal.v1.html`.
- Без клиппинга исходного SVG и без наложения «кусков» старого арта.
- Каждое щупальце: цепочка костей (`bones[]`) + процедурный skin (оболочка вокруг сплайна).
- Псевдо-3D: depth-слои (back/mid/front), разные амплитуды и фазы, маска объема через градиенты.

## Структура рига
- `Tentacle.update()`:
  - базовый угол,
  - волна + твист,
  - демпфирование к кончику.
- `Tentacle.sampleSpine()`:
  - Catmull-Rom интерполяция по узлам костей.
- `Tentacle.draw()`:
  - вычисление левого/правого контура с толщиной вдоль спайна,
  - заливка + контур + блики + присоски.
